Belajar Fundamental JS

1. Popup Alert

Alert biasa digunakan untuk menampilkan pesan dan untuk mendapatkan perhatian dari pengguna.

Contoh penerapan popup alert :
alert('Halo Selamat Datang di Arkatama')

2. Popup Prompt

Prompt sering digunakan jika kita ingin pengguna menginputkan sebuah nilai terlebih dahulu sebelum memasuki sebuah halaman.

Contoh penerapan popup prompt :
let string = prompt("Apakah kamu mau makan malam denganku?")
alert('Jawaban anda ' + string)

3. Popup Confirm

Confirm digunakan untuk menampilkan pesan konfirmasi kepada pengguna. Confirm box menampilkan pesan dengan dua buah tombol “OK” dan “CANCEL”.

Contoh penerapan popup confirm :
let string2 = confirm("Apakah kamu mau makan malam denganku?")
string2 ? alert('Jawaban anda mau') : alert('Jawaban anda tidak mau')
