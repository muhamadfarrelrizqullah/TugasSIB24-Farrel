Belajar Fundamental JS

1. Operator Aritmatika

Operator aritmatika adalah operator yang dapat digunakan untuk melakukan operasi aritmatika seperti penjumlahan, pembagian, perkalian dan sebagainya.

Contoh penerapan operator aritmatika :
let a = 6
let b = 3
var mainDiv = document.querySelector('.main .output')
mainDiv.innerHTML = "Penjumlahan: " + (a+b) + "Pengurangan: " + (a-b) + "Perkalian: " + (a*b) + "Pangkat: " + (a**b) + "Pembagian: " + (a/b) + "Modulus: " + (a%b)

2. Operator Perbandingan

Operator Relasi/ Perbandingan dapat digunakan untuk membandingkan dua nilai.Operator perbandingan akan menghasilkan salah satu dari nilai true atau false.

Contoh penerapan operator perbandingan :
let c = 4
let d = 7
var mainDiv = document.querySelector('.main .output2')
mainDiv.innerHTML = "Sama dengan: " + (c==d) + "Kurang dari: " + (c < d) + "Lebih dari: " + (c > d) + "Tidak sama dengan: " + (c!=d)

3. Operator Logika

Merupakan operator yang dapat digunakan untuk operasi terhadap dua nilai boolean.

Contoh penerapan operator logika :
let e = 3
let f = 5
var mainDiv = document.querySelector('.main .output3')
mainDiv.innerHTML = "And: " + (e&&f) + "OR: " + (e||f) + "Negasi: " + (!e)

4. Operator Bitwise

Operator bitwise merupakan operator yang digunakan untuk operasi berdasarkan bit (biner).

Contoh penerapan operator bitwise :
let g = 3
let h = 5
var mainDiv = document.querySelector('.main .output4')
mainDiv.innerHTML = "And: " + (g&h) + "OR: " + (g|h) + "XOR: " + (g^h) + "Negasi: " + (~g)

5. Operator Ternary

Operator ternary merupakan operator yang terdiri dari tiga bagian (kiri, tengah dan kanan).

Contoh penerapan operator ternary :
let i = 3
let j = 5
var mainDiv = document.querySelector('.main .output5')
mainDiv.innerHTML = "Hasil operator ternary: " + (i==j ? 'sama' : 'tidak sama')
