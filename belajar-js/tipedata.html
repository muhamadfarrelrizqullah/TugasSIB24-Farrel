Belajar Fundamental JS

1. Tipe Data Number

Data type atau tipe data dalam bahasa pemrograman adalah sebuah klasifikasi yang menentukan suatu tipe dari nilai. Tipe data number dalam Javascript digunakan untuk mempresentasikan integer (bilangan bulat) dan float (bilangan pecahan).

Contoh penerapan tipe data number :
let bulat =25
let desimal = 25.5
var mainDiv = document.querySelector('.main .output')
mainDiv.innerHTML = "Bulat: " + bulat + " Desimal: " + desimal + " Hasil: " + (desimal+bulat)

2. Tipe Data Bignt

Dalam Javascript tipe data number tidak bisa mempresentasikan bilangan bulat lebih dari (253-1) dengan kata lain bernilai 9007199254740991 atau untuk bilangan negatif - (253-1).

Contoh penerapan tipe data bigint :
let bulat2 =2515627288299283883993838930399399202933n
var mainDiv = document.querySelector('.main .output2')
mainDiv.innerHTML = "Bulat: " + bulat2

3. Tipe Data String

Tipe data string atau text adalah tipe data yang berisikan kumpulan kosong atau lebih karakter.

Contoh penerapan tipe data string :
let text = "Arkatama"
var mainDiv = document.querySelector('.main .output3')
mainDiv.innerHTML = "Text: " + text

4. Tipe Data Boolean

Tipe data boolean adalah tipe data yang berisikan data kebenaran, artinya hanya ada dua data, benar dan salah (yes atau no).

Contoh penerapan tipe data boolean :
let perbandingan = 1 > 7
var mainDiv = document.querySelector('.main .output4')
mainDiv.innerHTML = "Perbandingan: " + perbandingan

5. Tipe Data Undefined

Merupakan nilai khusus yang merepresentasikan bahwa sebuah variabel belum memiliki nilai yang di-assign terhadapnya.

Contoh penerapan tipe data undefined :
let admin
var mainDiv = document.querySelector('.main .output5')
mainDiv.innerHTML = "Admin: " + admin

6. Tipe Data Null

Null merupakan sebuah nilai khusus yang bukan milik tipe data apapun.

Contoh penerapan tipe data null :
let admin2 = null
var mainDiv = document.querySelector('.main .output6')
mainDiv.innerHTML = "Admin: " + admin2

7. Tipe Data Symbol

Tipe data symbol digunakan untuk sesuatu yang bernilai unik dan tidak dapat diubah (immutable).

Contoh penerapan tipe data symbol :
let employee = Symbol('joko')
var mainDiv = document.querySelector('.main .output7')
mainDiv.innerHTML = "Karyawan: " + employee.description

8. Tipe Data Object

Tipe data khusus yang terdiri dari pasangan kunci dan nilai (key-value pair).

Contoh penerapan tipe data object :
let employee2 = {
name: "john",
age: 25,
job: "Fullstack Web Dev"
}
var mainDiv = document.querySelector('.main .output8')
mainDiv.innerHTML = "Nama: " + employee2.name + " Umur: " + employee2.age + " Job: " + employee2.job
