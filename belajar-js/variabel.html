Belajar Fundamental JS

1. Variabel

Variabel adalah sebuah penyimpanan dengan nama (named storage) untuk menampung data. Dengan menyimpan data di variable, kita bisa menggunakannya lagi dengan menyebutkan nama variabel nya. Untuk membuat variable di JavaScript, kita bisa menggunakan kata kunci var diikuti dengan nama variabel nya.

Contoh penerapan variabel :
var fullName = 'Muhamad Farrel Rizqullah'
var age = 20
var mainDiv = document.querySelector('.main .output')
mainDiv.innerHTML = "Nama: " + fullName + "Umur: " + age

2. Mengubah Value Variabel

Setelah variable di deklarasikan, kita bisa mengubah value atau nilai di dalam variabel tersebut. Untuk mengubahnya, kita bisa menggunakan perintah nama variable diikuti dengan tanda = (sama dengan) lalu diikuti dengan value atau nilai nya.

Contoh penerapan mengubah value variabel :
var fullNameNew = 'Muhamad Farrel Rizqullah'
fullNameNew = 'Farrel Rizq'
var mainDiv = document.querySelector('.main .output2')
mainDiv.innerHTML = "Nama Baru: " + fullNameNew
