Belajar Fundamental JS

1. Tipe Konversi String

Konversi ke tipe data String biasanya diperlukan saat membutuhkan nilai yang berbentuk String.

Contoh penerapan tipe konversi string :
let value = false
let string = String(value)
var mainDiv = document.querySelector('.main .output')
mainDiv.innerHTML = "Value: " + value

2. Tipe Konversi Numeric

Sebagai contoh jika kita melakukan operasi pembagian kepada bilangan non-numeric maka akan dihasilkan tipe data numeric.

Contoh penerapan tipe konversi numeric :
let str = "123"
let nbr = Number(str)
var mainDiv = document.querySelector('.main .output2')
mainDiv.innerHTML = "Number: " + nbr

3. Tipe Konversi Boolean

Konversi Boolean memiliki aturan sebagai berikut: Nilai 0, String yang kosong, null, undefined, and NaN, menjadi false. Selain itu menjadi true.

Contoh penerapan tipe konversi boolean :
var mainDiv = document.querySelector('.main .output3')
mainDiv.innerHTML = "Boolean(0): " + Boolean(0) +" Boolean(1): " + Boolean(1)
